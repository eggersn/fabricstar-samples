# README
## Prerequisite
Make sure the Docker image of Fabric* Kafka is built on the machine.

## Run
To start the sample network and run the example chaincode execute the `run_all.sh` script in the respective `test_network` folder.

## Stop
To stop the sample network use `./network.sh down` in the respective `test_network` folder.

## Troubleshooting
Known issue for Docker on macOS: https://github.com/docker/for-mac/issues/4955
